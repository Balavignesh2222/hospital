Bubble Sort Algorithm

            1. Start
            2. Set n = length of array
            3. For i = 0 to n-1:
                a. For j = 0 to n-i-2:
                    i. If array[j] > array[j+1]:
                        - Swap array[j] and array[j+1]
                        - Highlight the swapped elements
                    ii. Otherwise:
                        - Do nothing
            4. End
